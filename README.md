# Promethues
一款以古希腊为背景的休闲小游戏，采用AndEngine引擎做的一个游戏😂😂😂😂😂。

背景故事就是，普罗米修斯偷了圣火，然后需要把圣火扔给人类，中间有一系列道具，玩家就需要帮助普罗米修斯把生活准确的扔到人类手上。
